# App Store Metadata

## 推广文本（中文）

在菜单栏实时查看网络状态，快速完成诊断、测速、异常提醒和截屏，帮助你更快定位并处理网络问题。

## Promotional Text (English)

Monitor network status from the menu bar and quickly access diagnostics, speed tests, alerts, and screenshots when needed.

## 描述（中文）

vm-net 是一款面向 macOS 的菜单栏网络工具。

它可以在菜单栏中显示当前上传和下载速率，并提供网络活动查看、网络诊断、网络测速和截屏等功能，方便你在需要时快速切换和使用。

适合希望在桌面上常驻查看网络状态，或在网络变慢时做基础排查的用户。

## Description (English)

vm-net is a menu bar network utility for macOS.

It shows current upload and download rates in the menu bar, and provides network activity, network diagnostics, speed testing, and screenshot tools so they are easy to access when needed.

It is suitable for people who want to keep an eye on network status from the desktop or do basic checks when the network feels slow.

## 此版本的新增内容（中文）

- 新增截屏快捷功能，支持快捷键触发、自定义区域选择和基础标注
- 优化截屏保存流程，支持选择保存位置或直接复制到剪贴板
- 重构主窗口结构，提升总览、网络活动、诊断和快捷功能页的使用效率
- 优化网络活动页与总览页布局，改进多语言环境下的窗口展示与缩放表现
- 收紧应用权限与发布配置，降低提审风险并提升发布一致性
