vm-net

macOS network utility

A simpler way to monitor network activity on your Mac.

vm-net keeps upload and download speed visible from the menu bar, adds one-click speed tests and diagnostics, and optionally turns network status into a lightweight desktop pet experience.

Privacy Policy GitHub Repository

Core capabilities

The product stays focused on a few practical workflows instead of trying to be a full network suite.

Real-time visibility

View live upload and download speed from the menu bar or a floating capsule.

Fast checks

Run a speed test or network diagnosis when you need to confirm connection quality.

Flexible presentation

Keep the interface minimal, or enable the optional desktop pet for a more visual mode.

Who it is for

vm-net is built for developers, remote workers, and Mac users who want a lightweight way to keep network performance in view without opening a larger dashboard.
